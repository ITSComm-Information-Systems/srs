Telephone Upgrade Project Transition Tools

Welcome to the Telephone Upgrade Project Transition Tools page. The apps below enable Telephone Upgrade Unit Ambassadors and SRS submitters to enter and update information needed to transition phone numbers to Zoom Phone.

Telephone Upgrade Request Form (TURF)

Submit telephone numbers to transition to Zoom Phone.

Pause Zoom Phone Transition

Temporarily pause softphones selected to transition to Zoom Phone.

Location Verification-Deskset

Verify location for telephone numbers scheduled to receive a deskset.

Add Deskset

Request a project-funded deskset for a number that has already transitioned to softphone only
